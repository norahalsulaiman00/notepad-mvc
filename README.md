# Notepad MVC

A Java desktop notepad application developed using the MVC (Model-View-Controller) design pattern.

## Features
- Create and edit text documents
- Save files
- Manage document settings
- Separate application logic using MVC architecture

## Technologies
- Java
- Java Swing
- MVC Architecture

## How to Run
Open the project in Eclipse or IntelliJ IDEA and run `NotepadApp.java`.
